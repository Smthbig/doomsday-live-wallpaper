☠️ Doomsday — Live Wallpaper Countdown App

A premium Android live wallpaper countdown app built to visualize time, goals, and yearly progress using elegant dot-based layouts.

Designed and developed by Krishna (DevKrishna) 🚀


---

✨ About the App

Doomsday turns your lock screen into a minimal progress visualization wallpaper.

Each dot represents progress through:

📅 Year mode → each dot = a day in the year

🎯 Goal mode → each dot = a day in your custom target

⚡ Auto mode → smart default yearly countdown


The current day is highlighted with a premium accent ring, giving a clean “time is running” aesthetic.
